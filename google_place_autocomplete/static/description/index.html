Google Place Autocomplete & Map

Watch the video

Google Place Autocomplete with Google Map Widget on Odoo backend views. Sync between map marker and address input value. When you change value at one, the other also change, so you can pick an address from the map by dragging the marker.

Use: <field name="name_of_field" widget="place_autocomplete" class="oe_inline"/>

Main Features

Place autocomplete.
Show/Hide map when click icon.
Sync changes between map and input.
Easy to use anywhere.

My Apps Published

Set google_maps_key_api:
Settings >> Active the developer mode
And
Settings >> Parameters >> System Parameters >> google_maps_api_key
Note: You need enable: Google Maps JavaScript API, Google Maps Geocoding API and Google Places API Web Service
