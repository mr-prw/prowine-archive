Nova Poshta Base

Nova Poshta Delivery: Synchronization of base catalogs

The module features

Catalogs of Nova Poshta delivery service:

  • Areas of Ukraine
  • Settlements of Ukraine
  • Cities of Nova Poshta
  • Warehouses

Periodic automatic synchronization of catalogs.

New delivery fields in a partner form.

How to use this module

CONFIGURATION

Go to the Nova Poshta menu and select the Configuration - Accounts submenu.

Then create a new account:

  1. Input the account name.
  2. Fill in the API key.
  3. To start synchronization immediately you need to click on Sync catalogs.
Synchronization may take a long time.

To configure Scheduled Action go to the Settings menu and select the Technical - Automation - Scheduled Actions submenu.

You need to activate Developer Mode before.

Activate and adjust the action Nova Poshta: synchronization of catalogs.

USAGE

Go to a partner form and input the delivery information:

  • Select delivery city.
  • Select a delivery warehouse.
  • Click on Save.
