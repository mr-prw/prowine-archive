How to use

Configuration.

Here we added a new field in hr contact named salary per hour in employee contracts that can be used to calculate salary for worked(timesheet) hours.


Method for calculating.

We have Implemented new method getDuration() in our module that can be used in payroll salary rule. This method returns total duration of hours from timesheet activities for specific employee per payslip. It filters duration according to payslip 'date_from' and 'date_to'. So its used on the salary rules as described in image. This salary rules already added on our module so you don't need to manually add it, once you instanlled our module you will get it by default.


you can use the rule and create payslips according to that.
